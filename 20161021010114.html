使用Pelican创建你的个人博客

Posted by dreamChou24 on 五 21 十月 2016

Pelican是用Python实现的静态网站生成器。一般用于博客、新闻等静态内容网站制作，使用它，只需要懂得Markdown语法，以及Pelican自己的一些小命令。

当然，如果想要做成漂亮的网站，那么你需要找到Pelican的网站模板，使用它让实现酷炫清奇的网站。如果想要让网站能够在互联网上访问，那么你需要用到内容托管服务，例如七牛、github等，或者干脆购买一台云服务器。

安装Pelican很简单，使用命令pip即可，为了一个干净的环境的，这里建议使用virtualenv环境进行安装。

pip install virtualenv
virtualenv env
source env/bin/activate
#这时你处于虚拟环境,然后就是安装pelican,因为文章内容使用markdon语法,因此还需要安装python的markdown模块用于解析
pip install pelican
pip install markdown
mkdir myblog
cd myblog
pelican-quickstart
#执行以上命令用于生产pelican静态网站的目录及配置文件,它会问你几个问题，为了最简化，我大多数都选否
#目录及配置文件生成路径，默认就是当前路径
> Where do you want to create your new web site? [.] 
#网站标题
> What will be the title of this web site? xiaoming'blog
#网站作者
> Who will be the author of this web site? xiaoming
#语言，默认为英文
> What will be the default language of this web site? [en] 
#是否要为网站配置域名
> Do you want to specify a URL prefix? e.g., http://example.com   (Y/n) n
#是否提供分页功能
> Do you want to enable article pagination? (Y/n) y
#每页几条内容
> How many articles per page do you want? [10] 6
#时区
> What is your time zone? [Europe/Paris] 
#是否想要开启内容自动生成和发布功能
> Do you want to generate a Fabfile/Makefile to automate generation and publishing? (Y/n) n
#是否想要一个简单HTTP服务并且该HTTP服务可以自动更新修改的内容
> Do you want an auto-reload & simpleHTTP script to assist with theme and site development? (Y/n) n

以上结束后，就会生成两个目录和两个脚本，主要有用的就是content，output，pelicanconf.py三个文件。 首先我们先写一篇文章

Title: 你好，世界 
Date: 2010-10-03 10:20 
Modified: 2016-10-05 19:30 
Category: Python 
Tags: pelican, virtualenv 
Slug: my-first-content 
Authors: xiaoming 
Summary: 这是小明的第一个博客文章

这里使用markdown语法进行书写内容。

> 这是引用

[这是我的个人博客](http://hongday.net)

    def hello(): 
        print 'Hello, world!'

    hello()

将文章保存为myfirstcontent.md，保存在content/下。然后执行生成网页和开启http服务命令

pelican content
Done: Processed 1 article, 0 drafts, 0 pages and 0 hidden pages in 0.24 seconds.
cd output
python -m SimpleHTTPServer
erving HTTP on 0.0.0.0 port 8000 ...

这时候你就可以通过127.0.0.1:8000访问到你的博客了。

当然默认的主题不太好看，我们可以自己去找好看的。比如访问这个clean-blog。下载它的主题，然后将主题放在myblog下。

ls
clean-blog      content         output          pelicanconf.py  pelicanconf.pyc publishconf.py
cd clean-blog
ls
README.md      screenshot.png static         templates
cd ..
echo "THEME = 'clean-blog'" >> pelicanconf.py
pelican content
Done: Processed 1 article, 0 drafts, 0 pages and 0 hidden pages in 0.24 seconds.
cd output
python -m SimpleHTTPServer
erving HTTP on 0.0.0.0 port 8000 ...

访问127.0.0.1:8000，你会有惊喜。
